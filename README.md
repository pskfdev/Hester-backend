# Hester-API

เป็นโปรเจคที่จัดทำขึ้นเพื่อเป็นผลงานในการฝึกเขียนเว็บไซต์ในส่วนหลังบ้าน API โดยใช้ `Pure PHP`

และมีฟังก์ชั่นดังนี้

* Login / Logout
* CRUD User
* CRUD Product
* CRUD Category
* CRUD Blog


## language & library

* php
* php pdo
* php oop
* jsonwebtoken
* mySQL
